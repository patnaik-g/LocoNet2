# Agent Instructions

This document provides instructions for AI agents working on this repository.

## Naming Conventions

All Markdown documentation files (`.md`) must be named using SNAKE_UPPERCASE format (e.g., `MY_DOCUMENT.md`).

## LocoNet Specification Compliance

Before making any changes to the LocoNet message handling, you **must** consult the `docs/SPECIFICATION_COMPLIANCE.md` file. This file contains a matrix of the implemented LocoNet features and their status.

After implementing any new features or fixing any bugs related to the LocoNet protocol, you **must** update the `docs/SPECIFICATION_COMPLIANCE.md` file to reflect the changes. This is critical for maintaining accurate documentation of the library's capabilities.
